# Number Guessing Game

A simple Python command-line game where the player tries to guess a randomly generated number. The program provides feedback until the correct number is guessed.

## Features

- Validates user input to ensure only numbers are accepted
- Provides hints such as "Too High" or "Too Low"
- Simple and easy-to-use command-line interface
